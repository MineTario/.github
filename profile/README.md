# MineTario.de

Willkommen auf dem offiziellen GitHub von **MineTario.de**.

Hier sammeln wir den Code und die Projekte, die gerade entstehen, während wir unseren Minecraft-Server aufbauen.  
Vieles ist noch in Arbeit – manches experimentell, manches noch nicht schön, aber alles mit dem Ziel, es Schritt für Schritt besser zu machen.

---

## Worum es hier geht

MineTario.de ist ein Minecraft-Projekt im Aufbau.  
Wir entwickeln eigene Plugins und Systeme und lernen dabei selbst noch viel dazu.

Dieses GitHub ist:
- für unseren internen Code
- für kleine Tools und Experimente
- für Dinge, die später vielleicht mal „fertig“ sind
